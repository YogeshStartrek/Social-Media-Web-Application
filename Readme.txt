Steps to run this application.

1. Open Vs code
2. Click on terminal->new terminal
3. type "cd server" and press enter
4. type "npm i" and press enter
5. type "npm start" and press enter...your backend server has started running
6. click new terminal
7. type "cd client" and press enter
8. type "npm i" and press enter
9. type "npm start" and press enter...your frontend has started running
